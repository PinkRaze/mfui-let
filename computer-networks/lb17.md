# Лабораторная работа №17  
## Перенос данных из Windows 7 в Windows 10

**Цель работы:**  
Научиться переносить пользовательские файлы и параметры из Windows 7 в Windows 10 с помощью стандартного средства переноса данных Windows (Windows Easy Transfer) и ручного копирования.

**Оборудование:**  
- Компьютер с Windows 7 Professional (исходная система)  
- Компьютер с Windows 10 Pro (целевая система) – можно использовать виртуальную машину.

---

## Ход выполнения работы

### 1. Подготовка в Windows 7 (исходная система)

1. Войдите в систему Windows 7.  
2. Создайте папку **«Для переноса»** (For transferring) в любом удобном месте (например, на рабочем столе).  
3. Откройте **Блокнот**, создайте файл с текстом **«Со старого ПК»** и сохраните его в папке `Для переноса` под именем **«Data»**.

### 2. Запуск средства переноса данных Windows в Windows 7

1. Нажмите **Пуск → Все программы → Стандартные → Служебные → Средство переноса данных Windows**.

### 4. Копирование папки «Перенос» из Windows 7 в Windows 10

Скопируйте папку **«Перенос»**, созданную на рабочем столе Windows 7, в гостевую систему Windows 10 (через общие папки, сеть или внешний носитель).

---

### 5. Запуск средства переноса на Windows 10 (через migwiz)

В Windows 10 нет встроенного средства переноса, как в Windows 7. Поэтому нужно скопировать папку `migwiz` из состава Windows 7 (она находится в `C:\Windows\System32\migwiz`) на рабочий стол Windows 10.

1. Скопируйте папку `migwiz` и перенесите её на рабочий стол Windows 10.

## Контрольные вопросы

1. **Все ли файлы были успешно перенесены?**  
   Да, файл `Data` был перенесён полностью. При использовании средства переноса Windows Easy Transfer можно переносить также настройки программ, учётных записей и другие пользовательские данные.

2. **Какие способы миграции (переноса) данных вы можете назвать?**  
   - Средство переноса данных Windows (Windows Easy Transfer) – для версий до Windows 10.  
   - Ручное копирование файлов и папок через внешний носитель, сеть или облачное хранилище.  
   - Специализированные программы (PCmover, Laplink, и др.).  
   - Использование OneDrive или другого облачного сервиса для синхронизации.  
   - Перенос через общую папку на виртуальной машине или по сети.
