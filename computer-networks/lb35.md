# Работа с файлами и папками в ОС GNU/Linux с помощью файлового менеджера Midnight Commander

**Цель практикума** — познакомиться с утилитой Midnight Commander, которая входит в состав операционной системы GNU/Linux и используется для управления файлами и директориями.

## Рассматриваются следующие темы:
1. общее знакомство с Midnight Commander;
2. настройка внешнего вида отображения файлов и папок;
3. упорядочивание объектов при выводе на экран;
4. чтение и изменение файлов, создание папок;
5. копирование файлов и папок;
6. перенос файлов и папок;
7. стирание файлов и папок;
8. создание символических ссылок;
9. определение занятого и свободного места на диске;
10. соединение с FTP-сервером;
11. соединение с SFTP-сервером через SSH;
12. использование встроенной командной строки.

## Порядок выполнения работы

Практическая работа выполняется на компьютерах, где установлена свободно распространяемая операционная система Canonical GNU/Linux Ubuntu 12.04 LTS или Debian 7 с графической оболочкой GNOME.

### Начальная подготовка

Обратитесь к преподавателю для включения компьютера. Войдите в систему, используя выданные логин и пароль.

## 1. Первый запуск Midnight Commander

Откройте эмулятор терминала комбинацией клавиш `<Ctrl+Alt+T>` или через меню `Applications → Accessories → Terminal`. Наберите команду `mc` (от англ. midnight commander). Разверните окно программы на весь экран.

Изучите интерфейс программы и основные элементы управления.

Все результаты оформляйте в текстовом редакторе LibreOffice Writer (доступен через `Applications → Office → LibreOffice Writer`).

Сделайте снимок окна Midnight Commander и сохраните его в отчёте.

## 2. Настройка отображения файлов и папок

В окне Midnight Commander выполните следующие шаги.

- **Посмотрите структуру каталога в виде дерева.** Откройте выпадающее меню `Right`, выберите пункт `Tree` (или нажмите последовательно `<F9>`, `<R>`, `<T>`). Изучите структуру папки `/etc`. Зафиксируйте внешний вид окна снимком экрана в отчёте.
- **Узнайте подробности о выбранном элементе.** В меню `Right` выберите `Info` (либо нажмите `<F9>`, `<R>`, `<Ctrl+x>`, `<i>`). На левой панели отметьте любой каталог или файл — справа появится информация о нём. Сделайте скриншот.
- **Быстро просмотрите содержимое файла.** В меню `Right` выберите `Quick View` (или `<F9>`, `<R>`, `<Ctrl+x>`, `<q>`). На левой панели выберите сначала папку, потом файл (например, `/etc/fstab`). Содержимое отобразится на правой панели. Сохраните снимок окна.
- **Верните правой панели обычный режим списка файлов.** Для этого в меню `Right` выберите `File listing` (или `<F9>`, `<R>`, `<g>`). Сделайте скриншот.

## 3. Сортировка объектов при выводе

- **Упорядочьте объекты по размеру.** Зайдите в меню `Right` → `Sort order` (или нажмите `<F9>`, `<R>`, `<S>`). В появившемся окне выберите `Size` левой кнопкой мыши или клавишей `<Space>` (либо клавишей `<S>`), затем нажмите `OK` (или `<Enter>`). Перейдите в каталог `/etc` и прокрутите список вниз. В самом низу должен оказаться самый большой файл. Сделайте скриншот.
- **Упорядочьте объекты по времени изменения.** Аналогично откройте `Sort order`, выберите `Modify Time` (клавиша `<m>`), подтвердите. В `/etc` внизу списка будет файл с самой свежей датой изменения. Сохраните снимок.
- **Упорядочьте объекты по имени.** Снова откройте `Sort order`, выберите `Name` (клавиша `<N>`), нажмите `OK`. В каталоге `/etc` в конце списка окажется файл с именем, начинающимся на букву, близкую к концу алфавита (например, на "z"). Сохраните скриншот.

## 4. Просмотр и редактирование файлов, создание новых папок

- Для просмотра текстового файла (возьмите, например, `/etc/fstab`) выделите его и выберите в меню `File` пункт `View` (или нажмите `<F9>`, `<F>`, `<V>`, либо просто клавишу `<F3>`). Сделайте снимок экрана.
- Чтобы создать новый файл, перейдите в домашнюю папку и нажмите `<Shift+F4>`. Наберите любой текст в открывшемся редакторе, сохраните файл под произвольным именем (например, `newfile`) клавишей `<F2>`, затем закройте редактор клавишей


`<F10>`. Откройте этот же файл снова через `<F4>` (или через меню `File` → `Edit`). Сделайте скриншот. Закройте редактор `<F10>`.
- Создайте в домашнем каталоге папку `new_dir`. Нажмите `<F7>` (или меню `File` → `Mkdir`), введите имя `new_dir` и нажмите `OK`. Установите курсор на созданную папку и сохраните изображение окна.

## 5. Копирование файлов и папок

Midnight Commander удобен для массового копирования, перемещения и удаления. Для этого на одной панели (обычно левой) открывают исходную папку, а на другой (правой) — папку назначения. Обе панели должны быть в режиме `File listing`. Переключаться между панелями можно мышью или клавишей `<Tab>`.

Скопируйте всю папку `/etc` в `/tmp`. Откройте на левой панели корневой каталог `/`, а на правой — `/tmp`.

На левой панели выберите папку `/etc` и нажмите `<F5>` (или меню `File` → `Copy`). Появится окно параметров копирования. Обратите внимание на флажок `Preserve attributes` — он работает аналогично команде `cp -p`. Нажмите `OK` — начнётся копирование с индикацией прогресса. После завершения перейдите на правой панели в скопированную папку и сделайте скриншот.

## 6. Перемещение файлов и папок

Перемещение делается так же, как копирование. Откройте на обеих панелях каталог `/tmp`. Переместите только что скопированную папку `etc` с левой панели на правую, переименовав её в `etc_new`.

Для этого на левой панели выделите папку `etc` и нажмите `<F6>` (или меню `File` → `Rename/Move`).

## 7. Удаление файлов и папок

Удалите созданную папку `/tmp/etc_new`. Выделите её и нажмите `<F8>` (или меню `File` → `Delete`). Подтвердите удаление, выбрав `Yes`. В следующем диалоге для рекурсивного удаления всей папки выберите `All`. По окончании сделайте скриншот.

## 8. Поиск файлов и папок

Поиск запускается через меню `Command` → `Find file` (или последовательно `<F9>`, `<C>`, `<F>`). Можно искать по имени и по содержимому.

- Найдите все графические файлы с расширением `.png` в домашней папке. Перейдите в домашний каталог на одной из панелей, откройте окно поиска. В поле `File name:` введите `*.png`, снимите галочку `Search for content`. Нажмите `OK`. Когда появится надпись `Finished`, сделайте снимок окна с результатами поиска. Закройте окно результатов.
- Найдите строку `clock` внутри файлов каталога `/etc`. Откройте папку `/etc` на любой панели, запустите поиск. В поле `File name:` укажите `*`. Поставьте галочку `Search for content` и в поле `Content:` введите `clock`. Нажмите `OK`. После завершения поиска сохраните скриншот. Закройте окно.

## 9. Создание символических ссылок

Перейдите в папку `/tmp` на обеих панелях. Выберите любой файл и создайте на него символическую ссылку. Для этого вызовите меню `File` → `Symlink` (или нажмите `<F9>`, `<F>`, `<S>`). В открывшемся окне допишите к предлагаемому имени файла слово `_link` и нажмите `OK`. Обратите внимание, что перед именем ссылки появится символ `@` — так в Midnight Commander обозначаются символические ссылки. Сделайте скриншот.

Установите курсор на созданную ссылку — путь к исходному файлу отобразится в нижней строке активной панели.

## 10. Оценка занятого и свободного места

Midnight Commander показывает информацию о дисковом пространстве:

- В правом нижнем углу каждой панели отображается `Доступно / Всего` (в абсолютных величинах и процентах).
- Чтобы узнать размер, занимаемый папками, выберите в меню `Command` пункт `Show directory sizes` (или нажмите `<Ctrl>+<Space>`). Перейдите в `/etc`, отсортируйте файлы по размеру и включите показ размеров папок. Сделайте снимок окна.

## 11. Подключение к FTP-серверу

В Midnight Commander есть встроенный FTP-клиент. Подключитесь к серверу `ftp://mirror.yandex.ru`. Для этого откройте меню `Left` → `FTP link` (или `<F9>`, `<L>`, `<P>`). В появившемся окне введите адрес `ftp://mirror.yandex.ru` и нажмите `OK`. Содержимое публичной папки FTP-сервера отобразится на активной панели. Сохраните скриншот.

## 13. Использование встроенной командной строки

В нижней части окна Midnight Commander всегда есть приглашение командной строки. Его можно использовать для выполнения команд.

Выполните какую-нибудь команду и посмотрите результат. Введите `date`. Нажмите `<Ctrl+O>`, чтобы временно скрыть окно Midnight Commander. Обратите внимание на вывод текущей даты в командной строке. Сохраните скриншот. Нажмите `<Ctrl+O>` снова, чтобы вернуться в Midnight Commander.

## Контрольные вопросы

1. **Что нужно сделать, чтобы запустить программу Midnight Commander?**
   - Открыть терминал и ввести команду: `mc`
2. **Как посмотреть структуру каталога в виде дерева?**
   - Нажать `Tab` для переключения на правую панель, затем `Ctrl` + `\` (или `Alt` + `,`)
   - Или через меню: `Left` / `Right` → `Tree view mode`
3. **Каким образом вывести информацию о выбранном файле или папке?**
   - Выделить объект стрелками, затем нажать `Ctrl` + `x` → `q`
   - Или через меню: `Command` → `File info`
4. **Как организовать быстрый просмотр содержимого файла?**
   - Нажать `Ctrl` + `x` → `q` (Quick view)
   - Или выбрать режим панели: `Left`/`Right` → `Quick view`
5. **Как переключить активную панель в режим обычного списка файлов?**
   - Нажать `Tab` для выбора нужной панели, затем `Ctrl` + `\` (или `Alt` + `,`) → выбрать `Listing mode`
6. **Как отсортировать объекты по их размеру?**
   - Нажать `Ctrl` + `t` (или `Alt` + `t`) несколько раз, пока не появится сортировка по размеру
   - Или через меню: `Command` → `Sort order` → `Size`
7. **Как отсортировать объекты по дате последнего изменения?**
   - `Ctrl` + `t` несколько раз до сортировки по дате
   - Или: `Command` → `Sort order` → `Modification time`
8. **Как отсортировать объекты по имени?**
   - `Ctrl` + `t` до сортировки по имени
   - Или: `Command` → `Sort order` → `Name`
9. **Какие действия позволяют просмотреть файл во встроенном просмотрщике?**
   - Выделить файл и нажать `F3` (или `View`)
10. **Как создать новый файл?**
    - Нажать `Shift` + `F4`
    - Или через меню: `File` → `New file`
11. **Как отредактировать уже существующий файл во встроенном редакторе?**
    - Выделить файл и нажать `F4` (Edit)
12. **Каким образом создать новую папку?**
    - Нажать `F7`
    - Или через меню: `File` → `New directory`
13. **Как скопировать объекты с одной панели на другую?**
    - Выделить объекты (`Insert` для множественного выбора), затем нажать `F5`
14. **Как переместить или переименовать объекты, используя две панели?**
    - **Перемещение**: выделить объекты → `F6`
    - **Переименование**: выделить один объект → `F6` → изменить имя в диалоговом окне
15. **Как удалить файлы и папки?**
    - Выделить объекты → `F8` (или `Delete`)
16. **Какие шаги нужно выполнить для поиска объекта по имени?**
    - Нажать `Alt` + `?` (или `F9` → `Command` → `Find file`)
    - Ввести имя файла (можно с маской, например `*.txt`)
    - Нажать `Enter` для поиска
17. **Как найти определённый текст внутри файлов?**
    - В окне поиска (`Alt` + `?`) заполнить поле `Content` искомым текстом
    - Указать директорию и нажать `Find`
18. **Как создать символическую ссылку?**
    - Выделить файл/папку → `Ctrl` + `x` → `s` (Symlink)
    - Указать имя ссылки
19. **Каким способом узнать размер (занимаемое место) каталогов?**
    - Перейти в нужный каталог
    - Нажать `Ctrl` + `x` → `i` (Info)
    - Или выделить каталог и нажать `Ctrl` + `x` → `q`
20. **Как подключиться к FTP-серверу из Midnight Commander?**
    - В командной строке ввести: `cd /#ftp:user@server.ru:port/path`
    - Или через меню: `Left`/`Right` → `Shell link...` → ввести `ftp://user@server.ru`
21. **Как подключиться к SFTP-серверу через SSH?**
    - Ввести в командной строке: `cd /#sftp:user@server.ru:port/path`
    - Или через `Shell link...` → `sftp://user@server.ru`
22. **Как увидеть результат выполнения команды, введённой во встроенную командную строку?**
    - Ввести команду в нижней строке (после `>`)
    - Нажать `Enter`
    - Результат появится в окне просмотра (закрыть `Enter` или `F10`)
