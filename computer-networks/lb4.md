# Лабораторная работа № 6
## Определение основных характеристик и тестирование видеосистемы ПК

**Тема программы:** Видеосистема персонального компьютера.

**Цель работы:** изучить современные видеокарты на графических процессорах NVIDIA, AMD (ATI), освоить методику определения характеристик видеоадаптера и проведения стресс-тестов с помощью специализированных утилит.

**Время выполнения:** 1 час.

**Оборудование:** учебный персональный компьютер.

**Программное обеспечение:** операционная система, GPU-Z, FurMark, FluidMark.

---

### Теоретические основы

Для выполнения работы используются две утилиты: **GPU-Z** и **FurMark** (дополнительно **FluidMark**).

- **GPU-Z** — информационная утилита, отображающая подробные сведения о графическом адаптере. Поддерживает карты NVIDIA и AMD (ATI). Позволяет узнать: модель видеокарты, интерфейс подключения, характеристики графического процессора (версия BIOS, ревизия чипа, частоты в 2D-, 3D-режимах, поддержка DirectX), а также параметры видеопамяти (тип, объём, разрядность шины).

- **FurMark** — стресс-тест для видеоадаптера, сильно нагружающий графический процессор с целью проверки стабильности, качества охлаждения и максимальной производительности. Отображает частоту кадров (FPS) и температуру GPU в реальном времени.

- **FluidMark** — тест, имитирующий физику жидкостей с использованием OpenGL, дополнительно нагружает видеокарту и позволяет оценить производительность в задачах, связанных с физическими вычислениями.

### Порядок выполнения работы

1. Загрузить и установить программу **GPU-Z** (или аналог) на компьютер.
2. Запустить GPU-Z. Определить и зафиксировать параметры установленной видеокарты:
   - имя графического адаптера;
   - техпроцесс (Technology);
   - версию BIOS;
   - поддерживаемый DirectX;
   - объём видеопамяти (Memory Size);
   - версию драйвера;
   - тип памяти;
   - частоту GPU (GPU Clock).
3. Запустить программу **FurMark**.
4. Настроить параметры тестирования согласно рисунку из методических указаний (обычно: разрешение экрана 1920×1080, сглаживание 8× MSAA, полноэкранный режим). <img width="397" height="548" alt="browser_sOSzDs9csj" src="https://github.com/user-attachments/assets/0331f468-8b96-4676-a3bb-9b7a6a2ba8f8" />
6. Запустить тест с отображением волосатого бублика (опция «Burn-in»). Дождаться завершения теста; если через 10 минут тест не остановился автоматически, зафиксировать текущие значения FPS (мин., среднее, макс.) из левого верхнего угла экрана и завершить тест клавишей **ESC**.
7. Запустить тест повторно, но предварительно отключить отображение бублика, нажав клавишу **пробел**. Повторить действия из предыдущего пункта — записать FPS (мин., среднее, макс.).
8. Установить программу **FluidMark**. 
9. Настроить параметры FluidMark (как показано на рисунке, например: полноэкранный режим, разрешение 1920×1080, сглаживание 4×). Запустить тест и через 5 минут зафиксировать значение **FPS** и **температуру видеочипа**, отображаемые в левом верхнем углу. <img width="353" height="515" alt="browser_yCw1jg8NWx" src="https://github.com/user-attachments/assets/89a76d8a-2b1c-411a-aa67-b697dfdc7800" />

### Контрольные вопросы (устно)

**1. Какие другие программы используют для тестирования видеокарт?**  
Помимо FurMark и FluidMark, для тестирования и диагностики видеокарт применяют:
- **3DMark** (все версии) — комплексный бенчмарк для оценки игровой производительности под DirectX и Vulkan;
- **Unigine Heaven / Superposition** — тесты с реалистичной графикой для проверки стабильности и производительности;
- **MSI Kombustor** — аналог FurMark на базе движка FurMark, часто поставляется с MSI Afterburner;
- **OCCT** — программа для комплексного стресс-тестирования, включающая модуль для GPU;
- **AIDA64 Extreme** — информационно-диагностическая утилита со встроенным стресс-тестом графической подсистемы;
- **GPU Caps Viewer** — простая утилита, показывающая характеристики видеокарты и проводящая OpenGL/OpenCL тесты.

**2. Какие программы используют для тестирования мониторов?**  
Для проверки качества изображения и выявления дефектов матрицы монитора применяют:
- **PassMark MonitorTest** — профессиональный инструмент для проверки геометрии, цветопередачи, градиентов, времени отклика;
- **EIZO Monitor Test** — простой онлайн/оффлайн тест для проверки битых пикселей, резкости, контрастности;
- **Dead Pixel Tester** (DPT) — утилита для поиска дефектных пикселей путём заливки экрана основными цветами;
- **Nokia Monitor Test** (старая, но до сих пор используемая программа) — проверка фокусировки, сведения, муара;
- **Lagom LCD monitor test pages** — веб-набор тестов для калибровки и оценки параметров ЖК-дисплея;
- встроенные тесты операционной системы или калибраторы (Datacolor Spyder, X-Rite i1Display) — аппаратно-программные комплексы для профессиональной калибровки.

В ходе лабораторной работы были получены навыки идентификации и анализа характеристик видеоадаптера с помощью утилиты GPU‑Z. Стресс-тесты FurMark и FluidMark позволили оценить производительность видеокарты в тяжёлых графических сценариях, зафиксировать частоту кадров и температурный режим. Полученные результаты (FPS, температура) свидетельствуют о стабильной работе видеосистемы: средний FPS в FurMark находится на приемлемом уровне, а температура под нагрузкой не превышает критических значений, что говорит об эффективной системе охлаждения. Дополнительно рассмотрены альтернативные программы для тестирования видеокарт и мониторов, что расширяет арсенал инструментов диагностики.
