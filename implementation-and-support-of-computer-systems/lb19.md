# Лабораторная работа №19  
## Работа с журналами событий Windows

**Цель:** Изучить принцип работы журналов событий ОС Windows.

---

## Теоретические сведения

Даже когда пользователь ПК не совершает никаких действий, операционная система продолжает считывать и записывать множество данных. Наиболее важные события отслеживаются и автоматически записываются в особый лог, который в Windows называется **Журналом событий**. Основное предназначение Журнала событий — сбор данных, которые могут пригодиться при устранении неисправностей в работе системы, программного обеспечения и оборудования. Заносятся в него не только ошибки, но также предупреждения и успешные операции (установка программы, подключение к сети).

Физически Журнал событий представляет собой набор файлов в формате **EVTX**, хранящихся в системной папке `%SystemRoot%/System32/Winevt/Logs`. Эти файлы имеют бинарный формат, поэтому их нельзя открыть текстовым редактором. Для просмотра используется штатная утилита **eventvwr**.

Запустить утилиту можно:
- через Панель управления: **Администрирование → Просмотр событий**;
- командой `eventvwr.msc` в окне **Выполнить** (Win+R).

<img width="452" height="334" alt="browser_QFPX42LSUF" src="https://github.com/user-attachments/assets/fbaa62fd-915e-41b5-8d28-de21aa95c8f6" />

Основные разделы утилиты:
- **Журналы Windows** (основной раздел для работы):  
  - *Система* – события от драйверов и модулей Windows.  
  - *Приложения* – события от установленных программ.  
  - *Безопасность* – события входа, управления учётными записями, доступа к файлам и т.д.  
  - *Установка* – события установки обновлений и ПО.  
  - *Перенаправленные события*.
- **Журналы приложений и служб**.
- **Подписки**.

## Ход выполнения работы

### 1. Загрузка и запуск утилиты

1. Загрузитесь в Windows 7 или 10 на виртуальной машине.
2. Запустите **Просмотр событий** одним из способов:
   - Панель управления → Администрирование → Просмотр событий;
   - Нажмите **Win+R**, введите `eventvwr.msc`, нажмите **Enter**.

### 2. Изучение структуры журнала Windows

Ознакомьтесь с разделами: **Журналы Windows** → **Система**, **Приложения**, **Безопасность**, **Установка**. Изучите количество записей, уровни событий (Ошибка, Предупреждение, Сведения).

### 3. Фильтрация журнала «Система»

1. Выберите **Журналы Windows → Система**.
2. Нажмите справа **Фильтр текущего журнала** (или **Создать настраиваемое представление**).

## Контрольные вопросы

1. **Что такое журналы событий Windows?**  
   Это системные логи, в которые операционная система и приложения записывают информацию о важных событиях (ошибки, предупреждения, успешные операции) для последующего анализа и диагностики.

2. **Физическое расположение журнала Windows.**  
   Файлы журналов (в формате `.evtx`) хранятся в папке `%SystemRoot%/System32/Winevt/Logs`.

3. **Основные и дополнительные категории журнала Windows.**  
   - Основные: **Система**, **Приложения**, **Безопасность**.  
   - Дополнительные: **Установка**, **Перенаправленные события**.
