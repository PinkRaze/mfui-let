# «Основные команды оболочки операционной системы GNU/Linux — работа с файлами и каталогами»

**Цель занятия** — ознакомиться с имеющимися в операционной системе GNU/Linux командами оболочки для работы с файлами и каталогами.

## Изучаются следующие основные вопросы:
1) навигация по каталогам файловой системы и просмотр списка файлов и каталогов;
2) создание и удаление файлов и каталогов в файловой системе;
3) копирование и перемещение файлов в файловой системе;
4) копирование и перемещение каталогов в файловой системе;
5) использование шаблонов при работе с файлами и каталогами;
6) поиск файлов и каталогов;
7) создание и использование ссылок на файлы и каталоги;
8) оценка занятости свободного пространства.

---

## Задание

Практическое занятие проводится на компьютерах с установленной свободно-распространенной и бесплатной операционной системой Canonical GNU/Linux Ubuntu 12.04 LTS [3] или Debian 7 [4] с окружением рабочего стола GNOME.

Попросите преподавателя включить компьютер и авторизуйтесь в системе с помощью полученных учетных данных (логина и пароля).

Откройте окно терминала сочетанием клавиш `<Ctrl+Alt+T>` или из меню **Applications-Accessories-Terminal**.

- Выполните команду `[wq]` (англ. print working directory) для выхода пути текущего каталога. Сохраните вывод команд в отчет. Отчет оформляется в редакторе LibreOffice Writer.
- Разверните окно терминала на весь экран и выполните команду `[ls -a]` (англ. list directory contents) для просмотра содержимого текущего каталога. Сохраните вывод команд в отчет.

3. Копирование и перемещение файлов в файловой системе

Действия настоящего пункта выполняются внутри каталога /tmp.

- Перейдите в каталог /tmp с помощью команды `cd /tmp`. Сохраните результат выполнения команд в отчет.

Выполните копирование файла, как указано ниже.

- Скопируйте файл /etc/fsab4 в каталог /tmp с помощью команды `cp -r fsab4 /tmp`. Поверьте факт успешного копирования с помощью команды `ls -l /tmp/fsab4`. Получите информацию о исходном файле с помощью команды `ls -l /tmp/fsab4` и убедитесь, что дата и время создания исходного и скопированного файла не совпадают. Сохраните результаты выполнения команд в отчет.

- Скопируйте файл /etc/passwd в каталог /tmp с сохранением временных атрибутов с помощью команды `cp -p /etc/passwd /tmp` (англ. preserve). Проверьте факт успешного копирования с помощью команды `ls -l /tmp/passwd`. Получите информацию о исходном файле с помощью команды `ls -l /etc/passwd` и убедитесь, что дата и время создания исходного и скопированного файла теперь совпадают. Сохраните результаты выполнения команд в отчет.

**Контрольные вопросы**

1. Какую команду необходимо использовать для вывода текущего каталога?
   pwd

2. Какую команду необходимо использовать для вывода содержимого каталога?
   ls

3. Какую команду необходимо использовать для смены текущего каталога?
   cd

4. Какую команду необходимо использовать для создания нового каталога?
   mkdir

5. Какую команду необходимо использовать для удаления каталога?
   rmdir (для пустых) или rm -r (для непустых)

6. Какие команды необходимо использовать для создания пустого файла?
   touch или > filename

7. Как обозначается оператор перенаправления вывода?
   > (перезапись), >> (добавление)

8. Какую команду необходимо использовать для копирования объектов?
   cp

9. Какую команду необходимо использовать для перемещения объектов?
   mv

10. Какую команду необходимо использовать для удаления объектов?
    rm

11. Какую команду необходимо использовать для поиска объектов?
    find

12. Какую команду необходимо использовать для создания символических ссылок?
    ln -s

13. Какую команду необходимо использовать для получения занимаемого каталогом дискового пространства?
    du

14. Какую команду необходимо использовать для отображения занимаемого каталогом дискового пространства в наглядном виде?
    ncdu

15. Какую команду необходимо использовать для получения информации об использовании дискового пространства?
    df
